## Материалы

- [easy] [Основы C++](https://education.yandex.ru/handbook/cpp). Яндекс хендбук
- [easy] [Learn C++](https://www.learncpp.com/). Большой набор материалов по изучению C++
- [easy] [CMake Tutorial](https://cmake.org/cmake/help/latest/guide/tutorial/index.html). Оффициальный туториал по CMake
- [medium] [Просто о шаблонах С++](https://habr.com/ru/articles/599801/). Введение в метапрограммирование на C++
- [medium] [Программирование на C++, часть 1](https://compscicenter.ru/courses/cpp-1/2018-autumn/classes/). Качественные и крутые лекции. Не подходят для изучения с нуля
- [medium] [Программирование на C++, часть 2](https://compscicenter.ru/courses/cpp-2/2019-spring/classes/). Качественные и крутые лекции. Не подходят для изучения с нуля
- [medium] [Лекции C++](https://www.youtube.com/playlist?list=PLmGioT8z4uobADCfbspDyzZtshzFFCRyj). Продвинутые и короткие лекции по современным стандартам C++
- [hard] [Нескучное программирование. С++ без аллокаций памяти](https://stepik.org/course/251888/promo). Автор пишет много крутого материала на тему статического полиморфизма, думаю, что его курс также крут как и статьи.
- [easy] [Основы python](https://education.yandex.ru/handbook/python). Яндекс хендбук
- [easy] [Курс python](https://senjun.ru/courses/python/). Opensource курс по python
- [easy][en] [Python](https://www.geeksforgeeks.org/python-programming-language-tutorial/). Python Tutorial (geeksforgeeks)
- [medium] Андрей Александреску "Современное проектирование на С++". Хорошая книга по метапрограммированию
- [medium] Скотт Мейерс "Эффективный и современный С++. 42 рекомендации". Хорошая книга по C++11 и C++14
